## ⚙️ Setup Instructions

Follow these steps to run the project smoothly:

### 1️⃣ Create a Virtual Environment

First, create a virtual environment on your system (recommended for dependency management).

### 2️⃣ Open Terminal in VS Code

Open your project folder in VS Code, then launch the terminal.

### 3️⃣ Install Required Packages

Run the following command:

```bash
pip install SpeechRecognition pyttsx3 pyaudio
```

### ✅ Done!

All required dependencies are now installed and ready to use.
